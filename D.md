Criterio del confronto
$0 \leq f(x) \leq g(x), \forall x \in I$
se $\int_{I} g(x)dx < +\infty  \implies \int_{I}f(x)dx < +\infty$
se $\int_{I} f(x)dx = +\infty  \implies  \int_{I}g(x)dx = +\infty$

Criterio del confronto asintotico
$f(x) \geq 0, g(x)>0, \forall x \in I$
$\lim_{ x \to  x_{0}} \frac{f(x)}{g(x)} = l \neq 0, +\infty$.
$\int_{I}f(x)dx, \int_{I}g(x)dx$ hanno lo stesso comportamento.
se $l = 0$: se $\int_{I}g(x)dx < +\infty  \implies \int_{I}f(x)dx <+\infty$
se $l = +\infty$: se $\int_{I}g(x)dx \rightarrow +\infty \implies \int_{I}f(x)dx \rightarrow +\infty$.
